Отчет «Сводная ведомость»

Отчет предназначен для формирования сводной ведомости по лицевым счетам. В данном отчете выводятся сведения о помещениях лицевых счетов, а также сведения о начислениях на лицевые счета в разрезе услуг. Перед формированием отчета необходимо указать:

  • «Период» – указывается период, за который будет сформирован отчет;
  • «Организация» – из справочника «Организации» выбирается организация, по лицевым счетам которой будет сформирован отчет;
  • «Вид отчета» – указывается, нужно ли выводить в отчет сведения о недоплатах или переплатах.

Вверху формы расположены кнопки:

  • «Еще» – вызывается список стандартных действий;
  • «Сформировать отчет» – при нажатии будет сформирован отчет;
  • «Печать» – при нажатии будет выполнена печать отчета;
  • «Показать настройки» – открывается форма настроек параметров отчета. На форме дополнительно можно указать поля, которые будут отображены в таблице. На форме доступны следующие флаги:
    • «Льготы» - в отчет будут выводиться суммы льгот отдельной строкой, включая суммы корректировки по ним.
    • «Перерасчеты» - в отчет будут выводиться суммы корректировок отдельной строкой, включая суммы корректировки льгот.
    • «Пени» - в отчет будут выводиться суммы пени отдельной строкой.
    • «Повышающий коэффициент» - в отчет будут выводиться суммы начисления по услугам с учетом повышающего коэффициента.

Отчет вызывается из «Начисления услуг / Отчеты / Сводная ведомость».

Примечание

В настройках варианта отчета нельзя использовать группировку данных с типом группировки "Иерархия" и "Только иерархия", подробнее см. в разделе "Техническая информация".

Техническая информация:

Формирование отчета производится путем рекурсивного обхода дерева значений, полученного в результате выполнения запроса системы компоновки данных. Отчет может формироваться некорректно, если в настройках отчета используются поля с группировкой данных по иерархии, поэтому возможность их использования отключена.
